# コンプライアンス・リスク一覧（3サイト共通 / 非表示・注意推奨）

seijiselect プラットフォーム（国民 next-vote-match / 政治家 candidate-vote-match / 支援者 assistant-vote-match）の
**法令上リスクのある機能**と、**初期リリースで「表示しない・後回しにする」推奨**をまとめたもの。

> ⚠️ 本書は一般的な法令知識に基づく整理であり、法的助言ではありません。
> 寄付・献金・有償依頼・選挙期間に関わる機能を1つでも実装する前に、
> **選挙法・政治資金規正法に詳しい弁護士、および所轄の選挙管理委員会／総務省に必ず確認**してください。

---

## 1. もっとも重大なリスク：外国人寄付（政治資金規正法 22条の5）

- **外国人・外国法人・外国人が主な構成員の団体からの政治寄付は禁止**。
- **罰せられるのは「受け取った側（政治家・政治団体）」が中心**。寄付者だけの問題ではない。
- 「運営も政治家も外国人と特定できず知らずに受領」した場合、直ちに刑事責任とは限らない（故意犯）が、
  - その寄付は**違法・無効で返還対象**。
  - **本人確認・国籍確認をせず媒介したプラットフォーム自体の管理責任・レピュテーションリスク**が大きい。
- **→ 寄付を扱うなら「本人確認＋国籍確認（誓約含む）」が必須。用意できないうちは寄付決済機能を実装・表示しない。**

## 3. 「表示しない／後回し」推奨（設計段階で外す・封じる）

1. **寄付・献金の実決済**（全サイト）— 本人確認＋国籍確認の仕組みが整うまで非表示
2. **匿名寄付**の導線 — 作らない
3. **候補者本人へ直接送金**する導線 — 作らない（政党・後援会・資金管理団体経由のみ）
4. **企業・団体からの献金**導線 — 当面非表示
5. **有償の選挙運動**マッチング（assist）— 選挙運動依頼は扱わない／政策・地域活動に限定
6. **メールでの投票呼びかけ** — 一切作らない（中立通知のみ）
7. **候補者ランキング** — 後回し

## 4. 安全に進める原則

- **MVPは「お金を動かさない」**（見る・比較する・診断する・つながる相談まで）。決済・献金は別フェーズ＋リーガルレビュー必須。
- 支援者登録・寄付の入口で**国籍・属性（個人/法人/外国法人）の申告＋確認**を必須化（実装時）。
- **「選挙運動」と「政治活動／政策活動」をシステム上で区別**（選挙運動側は機能制限）。
- 各機能に**法令注意文**を表示（プロトに反映済み）。

## 5. プロトタイプへの反映状況（2026-06）

- **政治家トップ**：献金の禁止事項（外国人寄付・匿名/借名・本人直接送金・企業献金・選挙区内寄付/物品/飲食・メール選挙運動）を箇条書きで明記。
- **国民トップ**：中立性＋寄付（外国人・匿名禁止）＋電子メール選挙運動禁止を明記。
- **支援者トップ**：法令注意バナー（外国人支援・有償選挙運動・匿名寄付等）。
- **支援できること診断**：国籍・属性の確認、外国籍＋物品提供時の警告、有償業務は選挙運動を除く旨を表示。
